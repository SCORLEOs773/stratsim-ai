# StratSim AI

StratSim AI is an AI-powered strategic decision simulation engine that helps businesses evaluate different strategies using probabilistic modeling.

## Features

- Pricing strategy simulation
- Marketing investment simulation
- Monte Carlo risk analysis
- Strategy comparison engine
- AI-powered strategic recommendations
- FastAPI backend

## Tech Stack

- Python
- FastAPI
- Monte Carlo Simulation
- Pydantic
- REST APIs

## Example Use Case

Simulate multiple strategies:

- Increase product pricing
- Increase marketing budget

The system evaluates:

- Revenue impact
- Risk probability
- Strategic recommendation
